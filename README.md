# FLIPCORE V4 Modular CZ

Interní modulární aplikace v češtině pro správu skladu PC komponent, sestav, statistik, záloh a nastavení.

## Funkce
- dashboard s grafy
- modul komponent
- modul sestav
- modul statistik
- modul záloh
- modul nastavení
- izolovaná modulární struktura

## Spuštění
```bash
npm install
npm run dev
```
